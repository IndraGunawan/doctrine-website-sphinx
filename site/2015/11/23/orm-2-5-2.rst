Doctrine ORM 2.5.2 Release
==========================

We are happy to announce the immediate availability of Doctrine ORM
``2.5.2``.

Installation
------------

You can install this version of the ORM by using Composer and the
following ``composer.json`` contents:

.. code:: json


    {
        "require": {
            "doctrine/orm": "~2.5.2"
        }
    }
